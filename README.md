# micric
## Minimalistic `IDE` in top of `micro` text editor

# Some features :muscle:
- lsp support 🟦
- minimal memory usage 🟥
- filemanager 🟨
- terminal based 💚
- many colorschemes 🌈
- minimal memory usage 🐏
- simple shortcuts 🤬

# Dependencies ♒
- `micro` 
- `xclip` (*or `xsel`*)
- `fzf`
- `ctags`
